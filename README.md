## Getting started
For Ubuntu Linux with Ruby on Rails:<br />
* Clone the repo:
```
$ git clone https://github.com/jimmy2046/app_search
```
* Install the needed gems:
```
$ bundle install
```
* Create the database:
```
$ rails db:create
```
* Migrate the database:
```
$ rails db:migrate
```
* Start Rails server:
```
$ rails s -b 0.0.0.0 -p 3000
```
